                                                                                             
homework1
==============

Программа поисковик.
Пользователь вводит тему, стартовую точку (google.com, yandex.ru, ...), количество, рекурсивный поиск или нет.
Программа находит в интернете начиная от стартовой точки все ссылки на веб-странице и выводит в терминал.


ТРЕБОВАНИЯ
----------

Python 3.5.0 и выше;
Библиотеки: requests, beautifulsoup4.


ЗАПУСК
------

Для запуска программы из корня проекта homework1 выполните:
     python3 -m search <поизковый запрос> [--engine] {google|yandex} [--limit] <limit> [--depth] <depth>

Опции:
    --engine, -e - поисковик. По умолчанию google.
    --limit, -l - количество выводимых ссылок. Для google максимум 100, для yandex максимум 50.
                  Для большего надо делать обработку пагинации.
    --depth, -d - глубина рекурсии поиска ссылок

Пример:
     python3 -m search "язык программирования" -e google -l 10 -d 1
